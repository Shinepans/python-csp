Class Channel

CSP Channel objects.

In python-csp there are two sorts of channel. In JCSP terms these are Any2Any, ALTing channels. However, each channel creates an operating system level pipe. Since this is a file object the number of channels a program can create is limited to the maximum number of files the operating system allows to be open at any one time. To avoid this bottleneck use FileChannel objects, which close the file descriptor used for IPC after every read or write operations. Read and write operations are, however, over 20 time slower when performed on FileChannel objects.

Subclasses of Channel must call _setup() in their constructor and override put, get, __del__, __getstate__ and __setstate__, the latter two methods for pickling.
